1. create repo 
    gh repo create

2. add remote to repo
    git remote add "origin_name" main
    git add . 

3. Navigate to main branch
    git branch -M main

4. Commit files to branch
    git commit -m "initial commit"

5. push files to branch
    git push -u "origin_name" main


If having issues try to rebase.
    git pull --rebase
